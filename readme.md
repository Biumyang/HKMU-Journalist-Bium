COMP8090 – Data Structures, Algorithms and Problem Solving

## Task 1 – OOP Application

Flight Booking System implemented using Python.

Main classes:

- Passenger
- Flight
- Booking
- BookingSystem

Features:

- Create passenger
- Create flight
- Book flight
- Seat management

The system demonstrates Object-Oriented Programming concepts.
